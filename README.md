                        -ETape2 :nom et role des extensions 
WPvivid Backup

Rôle : Sauvegarde, restauration et migration. Cette extension permet de créer des copies de sécurité complètes de votre site (base de données et fichiers) pour prévenir toute perte de données. Elle est également très utilisée pour cloner un site ou le migrer vers un autre hébergeur.

Solid Security (anciennement connu sous le nom de iThemes Security)

Rôle : Sécurité globale du site. Comme l'indiquent ses sous-menus (Pare-feu, Vulnérabilités, Analyses de site), ce plugin est conçu pour protéger votre WordPress contre les cyberattaques (comme les attaques par force brute), bloquer les utilisateurs malveillants, imposer des mots de passe forts et surveiller l'état de santé sécuritaire du site.

LiteSpeed Cache

Rôle : Optimisation des performances et mise en cache. Ce plugin sert à accélérer considérablement le temps de chargement de votre site web. Il met les pages en cache (il est particulièrement performant s'il est hébergé sur un serveur web LiteSpeed) et offre des outils d'optimisation du code (minification CSS/JS/HTML).

EWWW Image Optimizer (visible dans le menu déroulant sous Outils)

Rôle : Compression et optimisation des images. Il réduit automatiquement le poids (en octets) des images que vous téléversez sur votre site, sans perte de qualité visible. Cela permet d'économiser de l'espace sur le serveur et d'améliorer la vitesse d'affichage des pages.
