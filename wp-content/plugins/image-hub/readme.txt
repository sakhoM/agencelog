Access and manage royalty-free images from Unsplash, Pixabay, Pexels, Openverse & Giphy without leaving your WordPress dashboard.

== Description ==

Whether you’re publishing blog posts, product pages, or marketing content, Image Hub gives you the visual tools to do it faster, safer, and better: no external tabs, downloads, or uploads. Plus, it's compatible with all the major website builders.

= Supported Image Providers =

- [Pexels](http://pexels.com)
- [Unsplash](http://unsplash.com)
- [Pixabay](http://pixabay.com)
- [Openverse](https://wordpress.org/openverse/)
- [Giphy](http://giphy.com)

== Key Features ==
- **Integrated stock search** from 5 royalty-free image sources
- **Direct import** to your WordPress media library
- **Safe for commercial use** – all sources have royalty-free options
- **Centralized asset management** – your own uploads + stock in one place
- **Performance-friendly** – lightweight, fast, and built for daily use


== External Services ==
Image Hub routes all API requests to third-party image providers (like Pexels, Unsplash, Pixabay, Openverse, and Giphy) **through a secure proxy at** [api.wpimagehub.com](api.wpimagehub.com). This architecture:

- **Keeps API keys private**
- **Delivers a standardized image format**
- **Optimizes response data for WordPress display**

The result? A unified, fast, and secure image search experience tailored to the WordPress editor.
